## Quickstart

1. Install MongoDB and ensure it is running.
2. use "git clone https://github.com/rajatsah95/chandniTravel.git" to clone git repository.
3. Setup backend:
   - `cd backend`
   - `npm install`
   - `npm run dev`
4. Setup frontend:
   - `cd frontend`
   - `npm install`
   - `npm run dev`
